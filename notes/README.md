# Notes

Short research notes. Faster to write than a full arc README, more substantive than a social-media reply. Use this directory for standalone ~500–1500 word pieces that answer a specific question or document a specific observation.

Each note is independent. If a note grows into a sustained inquiry, it should graduate to a dedicated arc directory (like `drift-indicators/` or `the-medium/`).
